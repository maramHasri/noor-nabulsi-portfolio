# Noor Portfolio

A modern, dark-themed portfolio website built with React and Vite.

## Features

- **Hero Section**: Animated introduction with avatar and tagline
- **Professional Summary**: Typing effect animation for the main heading
- **Work Experience**: Grid layout showcasing projects with interactive cards

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- React 18
- Vite
- CSS3 (Custom Properties, Animations, Grid, Flexbox)
